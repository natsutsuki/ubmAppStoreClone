# ubmAppStoreClone
仿AppStore转场动画

本案例涉及到的基础知识点:
UIViewContorller的构建和UICollectionView的使用
你可以参考这个视频:
https://www.youtube.com/watch?v=hexuJ6qL06I

更灵活，但比UITableView略微复杂。
